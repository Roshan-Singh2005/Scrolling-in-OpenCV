This project is a Python-based computer vision application that allows you to control mouse scrolling using simple hand gestures.

**Built with:**
* OpenCV
* MediaPipe
* PyAutoGUI

## âœ… Features

* **Touchless Control:** Scroll web pages, PDFs, and documents without touching the mouse.
* **Gesture Recognition:** Uses Google's MediaPipe for accurate hand tracking.
* **Stealth Mode:** Can run in the background (hidden window) while still detecting gestures.
* **Global Kill Switch:** Press 'q' at any time to terminate the program, even if the camera window is hidden.
* **Smooth Logic:** Prioritizes specific gestures to prevent accidental scrolling while moving your hand.
